# App

======================================================================================

## ⚙️ Requisitos

Antes de ejecutar el proyecto, asegúrate de tener instalado:

- Java JDK 17 o superior
- Apache NetBeans (recomendado)
- Conexión a internet (para descargar dependencias de Maven)

======================================================================================

1. Descargar el proyecto desde GitHub (ZIP o clone)
2. Descomprimir el archivo (si es ZIP)
3. Abrir NetBeans
4. Ir a: `File → Open Project`
5. Seleccionar la carpeta del proyecto (donde está el `pom.xml`)
6. Esperar a que Maven descargue las dependencias
7. Ejecutar el proyecto con `Run (F6)`

======================================================================================

## Notas importantes

- NO borrar la carpeta `src`
- NO borrar el archivo `pom.xml` (es esencial para Maven)
- La carpeta `target` se genera automáticamente y no debe modificarse
- La primera ejecución puede tardar mientras Maven descarga dependencias
